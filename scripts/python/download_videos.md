### Install

```
pip install -U yt-dlp
```

### Use

ba = best quality

### Site

```
yt-dlp --format "bv*+ba/b" https://www.website.com
```

### With Cookies

```
yt-dlp --cookies-from-browser safari https://www.website.com/
```
