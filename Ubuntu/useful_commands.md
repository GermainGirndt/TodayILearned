## Useful Commands


#### Machine Name

uname -a


#### SecureBoot
```
sudo mokutil --enable-validation

sudo mokutil --disable-validation
```

#### Extracting g tar - verbose
```
tar xvzf FILENAME.tar.gz 
```

#### Installing .sh file

* **Give write permission**
```
chmod +x /path/to/yourscript.sh
```

* **Run file with './'**
```
./yourscript.sh
```
and **NOT**

```
yourscript.sh
```
